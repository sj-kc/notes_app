
# Notes app

The notes app its CRUD application using ES6 Javascript syntax and localStorage.

### Clone the repo

```bash
git clone https://github.com/sungkimcastro/notes_app.git
```

### Install dependencies

```bash
yarn install
```

### Start a local server to preview the site

```bash
yarn dev-server
```
